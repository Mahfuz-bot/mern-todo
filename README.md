# Todo App

A simple Todo App built with React, allowing users to add, edit, and delete tasks. This app also demonstrates the gradual rendering of tasks with a delay effect, making the UI more engaging.

## Features

-  Add new todos
-  Edit existing todos
-  Delete todos
